# 🌐 Network Concepts

## 📌 Introducción

El networking está presente en casi toda la tecnología actual: dispositivos, software, Internet, etc.  
Gran parte de esto funciona gracias al modelo **TCP/IP**.

En este módulo veremos:

- Modelo OSI
- Modelo TCP/IP
- Protocolos de red
- Métodos de transmisión

---

# 🧱 Modelo OSI

El modelo **OSI (Open Systems Interconnection)** divide la comunicación en **7 capas**.

## 📷 Capas del modelo OSI

![Modelo OSI](./images/osi-model.png)

## 🔢 Capas del OSI

1. **Physical (Física)**
   - Transmisión de bits
   - Ej: cables, hubs

2. **Data Link (Enlace de datos)**
   - Comunicación entre nodos
   - Usa direcciones MAC

3. **Network (Red)**
   - Enrutamiento de paquetes
   - Usa direcciones IP

4. **Transport (Transporte)**
   - Comunicación extremo a extremo
   - Protocolos: TCP / UDP

5. **Session (Sesión)**
   - Maneja conexiones entre aplicaciones

6. **Presentation (Presentación)**
   - Traducción, compresión, cifrado

7. **Application (Aplicación)**
   - Interacción con el usuario
   - Protocolos: HTTP, FTP, SMTP, DNS

---

## 📤 Ejemplo: Enviar un archivo

1. Aplicación inicia envío  
2. Presentación cifra datos  
3. Sesión establece conexión  
4. Transporte divide datos  
5. Red enruta paquetes  
6. Enlace encapsula  
7. Física transmite bits  

---

# 🌐 Modelo TCP/IP

Modelo práctico usado en Internet (4 capas).

## 📷 Modelo TCP/IP

![Modelo TCP-IP](./images/tcp-ip-model.png)

## 🔢 Capas

1. **Link Layer**
   - Hardware y conexiones físicas  
   - Ethernet, Wi-Fi  

2. **Internet Layer**
   - Direccionamiento y rutas  
   - IP, ICMP  

3. **Transport Layer**
   - Comunicación entre dispositivos  
   - TCP, UDP  

4. **Application Layer**
   - Servicios al usuario  
   - HTTP, FTP, SMTP  

---

## ⚖️ OSI vs TCP/IP

| OSI (7 capas) | TCP/IP (4 capas) |
|--------------|-----------------|
| Más teórico | Más práctico |
| 7 niveles | 4 niveles |
| Detallado | Simplificado |

---

## 🌍 Ejemplo: Acceder a una web

1. Navegador usa HTTP  
2. TCP asegura envío  
3. IP enruta datos  
4. Red transmite

# 📡 Protocolos de Red

Los **protocolos** son reglas para la comunicación.

## 📊 Protocolos comunes

| Protocolo | Función |
|----------|--------|
| HTTP | Web |
| FTP | Transferencia de archivos |
| SMTP | Email |
| TCP | Comunicación confiable |
| UDP | Comunicación rápida |
| IP | Direccionamiento |

---

# 📡 Transmisión

## 🔄 Tipos

- **Analógica:** señal continua  
- **Digital:** bits (0 y 1)  

---

## 🔁 Modos

- **Simplex:** una dirección  
- **Half-Duplex:** dos direcciones alternadas  
- **Full-Duplex:** comunicación simultánea  

---

## 🔌 Medios de transmisión

### 🧵 Cableados
- Par trenzado (Ethernet)
- Coaxial
- Fibra óptica

### 📶 Inalámbricos
- Wi-Fi
- Radio
- Satélite
- Infrarrojo

# 🎯 Conclusión

- El modelo **OSI** ayuda a entender la red  
- El modelo **TCP/IP** es el que realmente se usa  
- Los protocolos permiten la comunicación  
- La transmisión define cómo viajan los datos  

👉 Todo esto es clave para **ciberseguridad**, porque entender cómo funciona la red = saber cómo atacarla y defenderla.
