# 📱 Mobile Security

## 🧠 ¿Qué es Mobile Security?

La **Seguridad Móvil** se enfoca en proteger:

- 📱 Dispositivos móviles (smartphones, tablets)
- 📂 Datos almacenados
- 🌐 Redes a las que se conectan

de amenazas como ataques, malware o accesos no autorizados.

---

## 🎯 Objetivo principal

- Proteger información sensible
- Evitar accesos indebidos
- Asegurar la **Confidencialidad, Integridad y Disponibilidad (CIA Triad)**

---

## 💰 Analogía: Cofre del tesoro

Imaginá que llevás un cofre con tus objetos más valiosos:

- 📱 Dispositivo → Cofre  
- 📂 Datos → Tesoros  
- 🔐 Contraseña → Llave  
- 🕵️ Hackers → Ladrones  

👉 Tu celular contiene información crítica de tu vida personal y profesional.

---

## ⚠️ Riesgos comunes

- 🕵️ Robo de información
- 📲 Apps maliciosas
- 🌐 Redes Wi-Fi inseguras
- 🔑 Robo de credenciales
- 📉 Pérdida o robo del dispositivo

---

## 🛡️ Capas de seguridad en Mobile Security

### 🔐 1. Device Security (Seguridad del dispositivo)

Protege el acceso físico al dispositivo:

- Contraseñas / PIN
- Biometría (huella, rostro)
- Bloqueo automático
- Remote wipe (borrado remoto)

---

### 📂 2. Data Security (Seguridad de datos)

Protege la información almacenada:

- Encriptación de datos
- Backups seguros
- Prevención de pérdida de datos (DLP)

---

### 🌐 3. Network Security (Seguridad de red)

Protege la comunicación:

- Uso de VPN
- Protocolos seguros
- Evitar redes Wi-Fi públicas inseguras

---

### 📲 4. Application Security (Seguridad de aplicaciones)

Protege contra apps maliciosas:

- Instalación desde fuentes confiables
- Control de permisos
- Revisión de aplicaciones
- Desarrollo seguro

---

## 🔑 Importancia

Los dispositivos móviles:

- Contienen datos personales y financieros
- Acceden a sistemas empresariales
- Son objetivos frecuentes de ataques

👉 Son una puerta de entrada crítica para los atacantes.

---

## 👥 Responsabilidades

### 💻 IT Department
- Implementa soluciones de seguridad
- Gestiona redes y dispositivos

---

### 🛡️ CISO
- Define estrategias de seguridad
- Evalúa riesgos
- Asegura cumplimiento normativo

---

### 🔍 Security Teams
- Realizan testing
- Detectan vulnerabilidades

---

### ⚙️ IT Security Managers
- Supervisan operaciones
- Mantienen herramientas de seguridad

---

## 🔍 Buenas prácticas

- 🔒 Usar contraseñas fuertes
- 📲 Mantener el sistema actualizado
- 🚫 Evitar apps desconocidas
- 🌐 No usar Wi-Fi público sin protección
- 🔄 Hacer backups regularmente

---

## ⚠️ Punto clave

👉 Un dispositivo comprometido puede dar acceso total a tu información personal y laboral.

---

## 🚀 Conclusión

La **Mobile Security** es esencial en el mundo actual:

- Protege datos críticos
- Reduce riesgos de ataques
- Asegura el acceso seguro a sistemas

👉 Tu celular es uno de los activos más importantes que tenés: tratálo como tal.
