# 🌐 Network Security (Resumen)

## 🧠 ¿Qué es Network Security?

La **seguridad de redes** es una parte fundamental de la **Seguridad de la Información (InfoSec)** que protege:

- Redes
- Dispositivos
- Datos en tránsito

Su objetivo es prevenir accesos no autorizados, ataques y daños, tanto de **amenazas externas** como **internas**.

---

## 🏠 Analogía simple

Se puede comparar con el sistema de seguridad de una casa:

- 🔒 Protege puertas y ventanas → protege la red
- 💎 Protege objetos de valor → protege datos
- 🚨 Detecta intrusos → detecta ataques

---

## 🧩 Elementos clave de Network Security

| Elemento | Descripción |
|----------|------------|
| **Firewalls** | Filtran el tráfico entre redes confiables y no confiables |
| **IDS/IPS** | Detectan y bloquean actividades sospechosas |
| **VPNs** | Conexiones seguras y cifradas sobre redes públicas |
| **Control de acceso** | Autenticación y autorización de usuarios |
| **Cifrado** | Protege datos en tránsito y en reposo |

---

## 📬 Analogía del cartero

- 🪪 **Identificación del cartero** → Autenticación
- 🎒 **Bolsa cerrada** → Firewall
- 👀 **Vigilancia constante** → IDS/IPS
- 🚚 **Courier seguro** → VPN
- 🔐 **Sellos de seguridad** → Cifrado

👉 Todo esto asegura:
- Confidencialidad
- Integridad
- Disponibilidad

---

## ⚠️ Amenazas y riesgos

- Ransomware
- Robo de datos
- Espionaje
- Hacktivismo

📉 Consecuencias:
- Pérdidas económicas
- Daño reputacional
- Problemas legales
- Interrupciones operativas

🔺 El riesgo aumenta con:
- Cloud computing
- IoT
- Trabajo remoto

---

## 🛡️ Importante

Un **firewall no es suficiente** por sí solo.

👉 La seguridad efectiva requiere múltiples capas (defensa en profundidad).

---

## 👥 Responsabilidades

### 🧑‍💼 Equipo de seguridad de red

- Diseñar la seguridad
- Implementar controles
- Monitorear tráfico
- Responder incidentes

👑 Liderado por:
- **Network Security Manager**
- Reporta al **CISO**

---

### 🔍 Testing de seguridad

Realizado por:

- **Pentesters**
- **Ethical Hackers**

✔️ Simulan ataques reales  
✔️ Detectan vulnerabilidades  
✔️ Ayudan a mejorar la seguridad  

---

### 🏢 Otros roles involucrados

- **CISO:** Define estrategia
- **CIO / IT Manager:** Asigna recursos
- **Administradores de red:** Operación diaria
- **Analistas de seguridad:** Monitoreo
- **Compliance:** Cumplimiento legal
- **Gestión de riesgos:** Prioriza inversiones

---

## 🚀 Conclusión

La seguridad de redes es:

- Compleja
- Dinámica
- Esencial

👉 Es una de las principales defensas para proteger los activos digitales de una organización.
