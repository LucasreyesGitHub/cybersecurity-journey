# 🛡️ Junior Cybersecurity Analyst - Hack The Box

## 📌 Descripción

Este repositorio contiene mis notas, aprendizajes y práctica del path **Junior Cybersecurity Analyst** de Hack The Box.

El objetivo de este path es desarrollar habilidades fundamentales en:

- Análisis de seguridad
- Monitoreo de eventos
- Respuesta a incidentes
- Uso de herramientas de ciberseguridad

---

## 🎯 Objetivos del aprendizaje

- Comprender los fundamentos de la ciberseguridad
- Analizar logs de sistemas (Linux/Windows)
- Detectar amenazas y comportamientos sospechosos
- Investigar incidentes de seguridad
- Familiarizarse con herramientas SOC

---

## 🧠 Contenidos cubiertos

### 🔐 Fundamentos de InfoSec
